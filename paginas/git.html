Configuración

git config --global user.name "Nombre y apellido"

git config --global user.email CORREO@ELECTRONICO

Inicio

Creacion de repositorio: git init

Creacion de repositorio-clonado: git clone + url del repositorio en github

Fundamentos

Consultar estado repositorio: git status

Creacion de repositorio-clonado: git clone + url del repositorio en github

Añadir archivos: git add . o git add + nombre del archivo

Hacer un commit: git commit -m "texto"

Ver historial de cambios git log o git log --graph para ver el grafo

Ramas

Crear rama: git branch+ nombre rama

Ver ramas disponibles: git branch

Cambiar de rama: git checkout + nombre de la rama

Fusionar ramas: git merge + nombre de la rama

Eliminar rama git branch -d +nombre de la rama

Remoto

Enviar cambios a remoto(github) git push+ nombre rama

Enviar cambios a remoto y crear rama: git push -u

Traer cambios de remoto: git pull

Fusionar ramas: git merge + nombre de la rama

Eliminar rama git branch -d +nombre de la rama
